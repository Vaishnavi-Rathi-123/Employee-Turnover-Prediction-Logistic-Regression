# Employee Turnover Prediction using Logistic Regression

## Problem Statement
TalentCore Pvt. Ltd. wants to predict whether an employee is likely to leave the company
to reduce recruitment cost and talent loss.

## Dataset Description
The dataset contains employee-related features such as:
- Job Satisfaction
- Monthly Income
- Work-Life Balance
- Training Hours
- Annual Bonus
- Employee Turnover (Target)

## Approach
- Data preprocessing
- Feature engineering
- Baseline Logistic Regression model
- Regularization using L1 (Lasso) and L2 (Ridge)
- Model comparison

## Algorithms Used
- Logistic Regression
- Logistic Regression with L1 Regularization
- Logistic Regression with L2 Regularization

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report

## Conclusion
Regularized Logistic Regression improved model stability and performance.
